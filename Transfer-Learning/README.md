# Classification using Transfer Learning

* torchvision
* numpy
* os
* Matplotlib
* numpy
* cv2
* torchsummary
* tqdm

**Используемые алгоритмы и методы:**

* Создание класса Custom Dataset
* Transfer Learning на ResNet 50
* Заморозка весов
* Аугментация данных
